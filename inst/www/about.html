INFEST is a Shiny-app (user-friendly web application) written in R to process raw data from the EPG system and to perform statistical analysis on insect feeding behavior data. The annotated raw data files from the 3rd generation EPG can be simultaneously compiled to generate data tables of duration of waveform events, waveform event counts and sequential event counts.

The first data table generated by INFEST (Waveform Duration by insect - WDi) is editable, so that the user can set manually the levels of a factor (Group), if appropriate. The 'Group' column is automatically imported by the other generated data tables: Number of Waverform Events by insect - NWEi, Waveform Duration per Event by insect - WDEi.

The user can explore all the nonsequential variables from the duration and event count tables using exploratory statistical methods such as barplots, boxplots, biplots and inferential methods based on statistical ANOVA modeling.

If the number of files imported to INFEST is greater than the number of waveforms, it automatically generates principal component analysis based on singular value decomposition of standardized data (mean 0, variance 1). The result is shown in a biplot containing confidence ellipses for the Group means.

where 'mu' and 'sigma' represent the location and dispersion parameters, respectively; 'nu', in some of these models, is associated with the probability of zero in excess.

Likelihood ratio tests are carried out to test for the effect of 'Group'. Marginal means of 'Groups' are estimated and pairwise comparisons can be done using Tukey's method for type-I error rate control.

The sequential event count data can be graphically studied in a directional network plot.
