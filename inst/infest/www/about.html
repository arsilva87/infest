INFEST is a Shiny-app (user-friendly web application) written in R to: 1) process raw data files (of different extentions such as .ana, .txt) from both the DC and AC-DC EPG systems, and 2) perform statistical analysis on insect feeding behavior data. The annotated raw data files can be simultaneously compiled to generate numerical data on duration of waveform events (WDi, WDEi), waveform event counts (NWEi) and sequential variables.

The column 'Group' of the data table of response variables generated by INFEST is editable, so that the user can set manually the levels of a factor (a treatment), if appropriate.

The user can explore all the response variables using univariate and multivariate methods, from exploratory statistical methods such as barplots, boxplots, biplots to inferential methods based on statistical modeling.

If the number of files imported by INFEST is larger than the number of waveforms, it automatically generates principal component analysis based on singular value decomposition of standardized data (mean 0, variance 1). The result is shown in a biplot containing confidence ellipses for the Group means.

Likelihood ratio tests are carried out to test for the effect of 'Group'. Marginal means of 'Groups' are estimated and pairwise comparisons can be done using Tukey's method for type-I error rate control.

The sequential event count data can be graphically studied in a directional network plot.

INFEST can be installed as an R package to be executed locally. Open up the R console and execute the following command lines: 

if (!require("devtools", quietly = TRUE))
   install.packages("devtools")
devtools::install_github("arsilva87/infest")

If needed, let R to update the package dependencies during installation. Once it is finished, execute the following command line to open INFEST:

infest::infest()
