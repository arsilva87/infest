To work properly, INFEST requires recent versions of Rtools and pandoc to be previously installed.

INFEST's instalation is expected to automatically install all its dependencies. However, one can run a manual check. Here are all packages INFEST needs:

  • shiny
  • devtools
  • gamlss
  • emmeans
  • estimability
  • ggeffects
  • igraph
  • factoextra
  • shinythemes
  • htmltools
  • DT
  • rhandsontable
  • shinyjs
  • reshape2
  • plotly
  • rmarkdown
  • multcomp
  • multcompView
  • shinybusy

Try loading a package in R with 

library(gamlss)
If it fails, install it 
install.packages("gamlss")
After making sure all packages are installed, install INFEST: 
devtools::install_github("arsilva87/infest")
